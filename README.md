# BL_EFCore_Crud_User

Frontend styling and layout for the EF Core CRUD User project.  
This repository contains the UI layer (HTML/CSS) used to interact with the backend API.

## Features
- Clean and responsive UI  
- Simple forms for user management  
- Easy to integrate with any .NET backend  

## Technologies
- HTML
- CSS

## Usage
Clone the repository and open the HTML files in your browser, or integrate them into your .NET MVC / Razor / Blazor project.
